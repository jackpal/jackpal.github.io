# Repository Configuration: jackpal.github.io

This repository contains the source code for Jack Palevich's professional blog, "GrammerJack".

## Tech Stack
- **Static Site Generator**: Jekyll
- **Hosting**: GitHub Pages
- **Theme**: Hydeout (remote_theme: fongandrew/hydeout)

## Tone and Style
Refer to [GEMINI.md](file:///Users/jackpal/Developer/jackpal.github.io/GEMINI.md) for detailed guidelines.
- **Tone**: Professional, clear, optimistic, and useful.
- **Style**: Brief, direct, active voice. Respect the reader's time.

## Post Metadata Requirements
Every post must include:
- `date`: YYYY-MM-DD HH:MM:SS
- `tags`: Space-separated list of ~5 relevant tags.
- `description`: A one-sentence summary.

## Directory Structure
- `_posts/`: Markdown files named `YYYY-MM-DD-title.md`.
- `assets/posts/`: Images and other assets, prefixed by the post's filename.
- `_includes/`, `_layouts/`, `_sass/`: Jekyll templates and styling.
